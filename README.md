# Электронный тематический словарь архангельских говоров

Интерактивный словарь диалектной лексики Архангельской области с возможностью поиска, фильтрации и визуализации на карте.

[🔗 Электронный тематический словарь архангельских говоров](https://arkhdialect.narfu.ru/)

## О проекте

Проект реализован в рамках гранта Российского научного фонда № 23-28-01380 «Тематический словарь архангельских говоров с электронной поддержкой».

Словарь содержит диалектную лексику, сгруппированную по тематическому принципу. Сейчас выделены категории:
- Одежда, обувь, головные уборы, украшения, ткани
- Жилые и хозяйственные постройки и их части
- Традиционные северные блюда и напитки

## Основной функционал

- Поиск слов по алфавиту и частичному совпадению
- Фильтрация по:
    - тематическим подкатегориям
    - районам Архангельской области
    - частям речи
- Лексический атлас
